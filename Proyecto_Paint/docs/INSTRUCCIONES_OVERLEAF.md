```
proyecto/
├── informe.tex
└── img/

    ├── protocolo_ps2.png
    ├── protocolo_uart.png
    ├── datapath_uart.png
    ├── flujo_arduino_fpga.png
```

## Pasos en Overleaf

1. Crear un nuevo proyecto en Overleaf
2. Subir el archivo `informe.tex`
3. Crear una carpeta llamada `img`
4. Descargar cada imagen de las URLs anteriores y subirlas a la carpeta `img` con los nombres indicados
5. Compilar el documento
